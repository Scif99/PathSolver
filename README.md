# PathSolver

## Download
https://github.com/Scif99/PathSolver/releases/tag/v1.0


Controls
- 1: Place start node
- 2: Place end node
- Left-Click: Place walls
- Right-Click: Place grass (grass nodes have a cost of 5) 
- SPACE: Runs the algorithm
- R: Resets the grid
- T: Switches between step/full mode

Algorithms
- B: Switch to BFS
- A: A*
- G: Greedy BFS
- D: Djikstra
